欢迎来到 FuelPHP

FuelPHP 是一个简单，灵活，社区驱动的 PHP5 web 框架. 它因现有框架一些不如意而诞生，被社区开发者所支持. FuelPHP 非常轻便, 能工作于大多数服务器，其清晰的语法被大家所喜爱.

快速安装

如果你非常珍惜时间，想在几秒中就可以获取, 你可以使用我们的快速安装（quick installer）. 使用 curl 库, 你可以安装一个基本的 oil. 通过它可以安装一个完整 Fuel 程序.

# 从web快速安装 oil 
$ curl get.fuelphp.com/oil | sh
# 现在 oil 已经安装完成， 在 Sites 目录下创建一个 blog 工程
$ cd Sites/
$ oil create blog

更多信息, 参看 安装 页面.

手动安装

如果你无法使用快速安装方法, 你也可以选择手动安装 Fuelphp. 查看详细 安装 页面.
