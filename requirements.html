要求

Web 服务器
  • 大多数主流 web 服务器. Fuel 已经测试于 Apache, IIS and Nginx.
  • 工作于大多数主流操作系统，包括 *nix and Windows.
PHP 版本
  • 必须是5.3或以上版本
PHP 扩展

Fuel 使用基本PHP扩展. 为避免兼容性问题, Fuel 只使用在PHP官方文档中记录扩展. 如果使用 PECL 存放的扩展 (需要系统管理员手动安装), Fuel will use them if present, and work around it if not.

下面是使用扩展列表, 以及操作它们的类:

Extension Used in Comments
fileinfo Upload 文档.
注意在 Windows 平台, 你需要使用 DLL 手动安装.
mbstring n/a 文档.
如果不存在, 框架不支持多字节支持. 你可以通过检查 MBSTRING 常量来确定是否支持多字节.
mcrypt Crypt 文档.
如果不存在, PHPSecLib 可以代替它的基本功能.

推荐

  • Web 服务器 - Apache 运行于 *nix.
  • PHP 版本 - 5.3.2
  • mod_rewrite - 在 URL 中移除 index.php.
  • PHPUnit - 3.5.12
