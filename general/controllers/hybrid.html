混合控制器

混合控制器使一个简单的基本控制器具有 REST 控制器和模板控制器功能.

更多关于模板和 Rest 控制器介绍 Controller_Rest 或者 Controller_Template pages.

和其他控制器类一样, 该类也放置在 fuel/app/classes/controller 目录. 它们需要继承 Controller_Hybrid 类以及使用 "Controller_" 作为默认前缀. 以下是 "test" 控制器示例: 

class Controller_Test extends Controller_Hybrid
{
	// this will be called if you request /test/list from the URL
	public function action_list()
	{
		$this->template->content = 'Hello World!';
	}

	// this will be called when you do an ajax GET request for /test/list?foo=bar
	public function get_list()
	{
		$this->response(array(
			'foo' => Input::get('foo'),
			'baz' => array(
				1, 50, 219
			),
			'empty' => null
		));
	}
}
