配置应用程序

介绍

The Fuel framework is built with configuration over convention in mind. This results in a highly customizable base on which you can start building your application.

配置文件存放在 app/config 目录内. 基本的程序配置已经放置在 app/config/config.php. 根据设置环境加载配置文件, 详情查看 环境.

配置选项

键 类型 默认 描述
base_url string 
null
 程序基本URL. 可以是相对路径. 必须以斜线结尾. ('/foo/', 'http://example.com/')
url_suffix string 
''
 需要通过 Fuel 来自动添加的后缀. 如果后缀是扩展名, 确保包含点(.)号 ('.html')
index_file string 
'index.php'
 主引导文件名. 如果使用 mod_rewrite，将它设置位false或者移除.
controller_prefix string 
'Controller_'
 通过URI映射寻找到控制器类名时所使用的类名前缀. 如果你想自己定义或者使用不同于 app/classes/controller 文件夹，你需要修改它.
profiling bool 
false
 是否开启分析器.
caching bool 
false
 是否开启缓存.
cache_dir string 
APPPATH.'cache/'
 缓存存放目录. 此目录必须可写.
cache_lifetime int 
3600
 缓存生存时间，单位秒.
ob_callback callback 
null
 提供 ob_start() 回调函数, 设为 ob_gzhandler 开启 gzip 编码输出.
errors.continue_on array 
array(
)
 出现哪些可以错误时进行执行. 查看 错误处理.
errors.throttle int 
10
 最多显示错误条数. (防止内存不足)
errors.notices bool 
true
 是否显示警告信息.
language string 
'en'
 默认程序使用语言, 用于 语言类.
locale string 
'en_US'
 被 setlocale() 用于一些 php 所依赖, 设置为 false 取消设置.
语法结构根据每个操作系统有差异, Ubuntu 中需要使用 .utf8 作为编码后缀
encoding string 
'UTF-8'
 应用程序默认字符编码.
server_gmt_offset int 
0
 time()使用格林尼治时间戳秒偏移量.
仅用来改正服务器错误配置: time() 函数总是返回距 1970 00:00:00 1月1日 GMT 相差秒数.
default_timezone string 
'UTC'
 服务器时区.
log_threshold int 
Fuel::L_WARNING
 日志记录阈值. 为要记录日志信息级别, 或者是定义所要记录级别的数组. 查看可选值
log_path string 
APPPATH.'logs/'
 日志保存目录. 目录必须可写.
log_date_format string 
'Y-m-d H:i:s'
 日志使用日期/时间格式.
security.csrf_autoload bool 
false
 是否自动检查跨站请求伪造密钥. 查看跨站请求伪造详情.
security.csrf_token_key string 
'fuel_csrf_token'
 使用 $_POST 哪个值存取跨站请求伪造密钥.
security.csrf_expiration int 
0
 跨站请求伪造 cookie 生存时间. 高于0值被视为有效秒数.
security.uri_filter array 
array('htmlentities')
 使用哪些 php 函数过滤 URI.
security.input_filter array 
array()
 使用哪些函数过滤输入数组 ($_GET, $_POST and $_COOKIE). 可以设为 xxs_clean, 这会根据输入大小造成一定性能上的影响.
security.output_filter array 
array()
 使用哪些函数过滤输出. 可设为 xxs_clean, 这会根据变量大小造成一定性能上的影响..
security.auto_filter_output bool 
true
 是否自动编码 (htmlentities) 视图输出数据.
security.whitelisted_classes array() 
array(
'stdClass',
'Fuel\\Core\\View',
'Fuel\\Core\\ViewModel',
'Closure'
)
 传递对象给输出编码转换将会转换为字符串或者抛出异常，除非他们是这里的数组里的成员 unless they are instances of the classes in this array.
cookie.expiration int 
0
 cookie 有效生存秒数.
cookie.path string 
'/'
 设置 cookie 有效目录.
cookie.domain string 
null
 设置 cookie 有效域名.
cookie.secure bool 
false
 只通过安全连接发送 cookie.
cookie.http_only bool 
false
 仅能够使用 HTTP 使用cookie, 禁止 JavaScript 访问.
module_paths array 
array()
 模块目录. 未指定位置时添加一个模块使用.
always_load.packages array 
array()
 哪些包自动加载. 例如: array('package') 或者 array('package' => PKGPATH.'path/to/package')
always_load.modules array 
array()
 哪些模块自动加载. 例如: array('package'). 为使自动加载器(AutoLoader) 通过 URL 可以找到模块. 你必须在 module_paths 设置好
always_load.classes array 
array()
 哪些类自动加载初始化.
always_load.config array 
array()
 哪些配置文件自动加载. 将配置文件加载入组: array('config') 或者 array('filename' => 'group'). 如果你不想加分配文件放入组，你只需要设为 null: array('filename' => null).
always_load.language array 
array()
 哪些语言文件自动加载. 将配置文件载入组: array('lang') 或者 array('filename' => 'group'). 如果你不想加分配文件放入组，你只需要设为 null: array('filename' => null).
