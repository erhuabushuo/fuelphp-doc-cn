类

什么是类?

一个就是一个常规 PHP 类. 在 Fuel 中，它何其他所有的类一样，不需要继承任何东西，或者遵循任何命名约定.

class Session

将会从 app/classes/session.php 加载.

加载类

不像其他框架需要手动加载. 在引用他们时自动会被加载(Controllers, Models, 等等).

子目录中的类

Just like Controllers, 类必须大写字母开头其余小写字母命名, 下划线可以告知将类放置在子目录下.

Class Session_Driver

将会从 app/classes/session/driver.php 加载

类和命名空间

为确定加载哪个 PHP 文件, FuelPHP 针对命名空间处理和下划线规则是相似的. 也就是说当类在子目录下, 你可以混合使用命名空间以及下划线然针对你的需求.

就以一个类在 app/classes/core/system/messages.php 文件为例. 这个类在文件中可以定义为:

// 全局命名空间，完全通过下划线命名
class Core_System_Messages {}

// 结合命名空间和下划线
namespace Core;
class System_Messages {}

// 完全通过命名空间
namespace Core\System;
class Messages {}

第一个方法最通用最简单. 命名空间版本在类中结合使用 Use 语句是非常方便使用已定义类.

初始化类

允许在类被加载时自动执行某些任务, 就好像 __construct() 初始化实例. 你可以通过在类创建一个公共静态(static) _init() 方法.

class Example {
	public static function _init()
	{
		// 类加载时调用
	}
}

如果加载者类中调用 _init() 方法 并且欲加载父类此方法, 除非你手动指定 parent::_init();，否则不会自动调用父类初始化方法
