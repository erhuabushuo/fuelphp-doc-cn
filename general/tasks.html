作业

什么是作业?

作业是可以通过命令运行或者设置cron job运行的类. 它们通常用来后端执行，时间作业，持续作业. 作业可以访问模型和他们类，就和控制器一样。.

创建作业

放置于 fuel/app/tasks 目录. 以下是作业 "example" 示例: 

namespace Fuel\Tasks;

class Example
{

	public function run($message = 'Hello!')
	{
		echo $message;
	}
}

使用oil refine工具调用:

$ php oil refine example "Good morning"
.

命令中使用作业名, 自动调用方法 "run()" .

分类作业为多个方法

你可以在你的作业中添加多个方法来达到一组作业.

public function current_date()
{
	echo date('Y-m-d');
}

使用如下方法访问:

$ php oil refine example:current_date
